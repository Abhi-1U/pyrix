### Alpha Edition Installation
> **Installation(from git):**  

```bash
$ git clone <this-repo>    
$ cd pyrix  
$ git checkout alpha  
$ pip install .  
```  
\* some operating systems see pip for Python2 and pip3 for Python3 ,in that case  
```bash
$ pip3 install .
```
